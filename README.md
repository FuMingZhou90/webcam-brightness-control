# Webcam Brightness Control 💡

A smart desktop utility that automatically adjusts your monitor's hardware brightness using your webcam's ambient light readings—giving your desktop setup the seamless, automatic brightness features of a laptop, tablet, or smartphone.

[![Support on Patreon](https://img.shields.io/badge/Support-Patreon-FF424D?style=for-the-badge&logo=patreon)](https://www.patreon.com/Jim9446)

---

## 🚀 Features
* **True Hardware Adjustments:** Controls your actual monitor brightness directly instead of relying on generic software overlays or messing with in-game contrast settings.
* **Smart Ambient Detection:** Uses your connected webcam to scan light levels and smooths out transitions cleanly.
* **Minimal Footprint:** Runs quietly in the background without hogging system resources.

## 📥 How to Install & Run
1. Head over to the **[Releases](https://github.com/FuMingZhou90/webcam-brightness-control/releases)** section on the right side of this page.
2. Download the latest `SmartBrightness.exe` file from the assets list.
3. Move the file to a preferred folder on your PC and double-click to run it!
